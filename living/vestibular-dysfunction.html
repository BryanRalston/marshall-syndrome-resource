Vestibular Dysfunction: The Hidden Balance Problem

Key Finding: A landmark NIH study (Griffith 2000) found measurable vestibular dysfunction in Marshall syndrome patients who had no balance complaints. The deficit was subclinical — detected only through formal testing. This means vestibular problems in Marshall syndrome are likely widespread but almost never screened for.

COL11A1 in the Inner Ear

  • COL11A1 is expressed in the tectorial membrane of the cochlea
  • Mouse models show significantly reduced vestibular stereocilia (hair cells that detect motion)
  • Both cochlear and vestibular structures develop from the same otic placode
  • If COL11A1 affects hearing (well-documented), it must also affect vestibular function
  • The vestibular organs use related but different proteins (cupulin in semicircular canals, otolin-1 in otoconia) — so the mechanism is indirect/developmental, not direct structural like in the cochlea

The Griffith Study — What We Know

  • 3 family members with Marshall syndrome (COL11A1 splice site mutation)
  • Progressive sensorineural hearing loss (cochlear origin)
  • Asymptomatic central and peripheral vestibular dysfunction on electronystagmography
  • Bony labyrinth was normal on imaging — the problem is in the membranous labyrinth and CNS
  • “Asymptomatic” = the brain compensates using vision and proprioception
  • Compensation fails under: fatigue, darkness, uneven surfaces, or if cervical proprioception is also compromised

The Multi-System Balance Problem

System Mechanism Evidence
Vestibular (inner ear) Reduced stereocilia, subclinical dysfunction Published (Griffith 2000, mouse models)
Proprioceptive (cervical) Cervical spine pathology disrupts neck proprioceptors Strong mechanistic
Motor (vestibulospinal) If myelopathy present, vestibulospinal tract compressed Anatomical certainty if cord compression
Autonomic (vestibulosympathetic) Degraded vestibular input to autonomic regulation Moderate, extrapolated from EDS
Important Distinction: “Legs giving out” could represent vestibulospinal tract dysfunction (sudden loss of extensor tone), corticospinal tract weakness, vestibular drop attacks, or orthostatic hypoperfusion — or a combination. The distinction matters because treatments differ.

Vestibular vs Cervicogenic Dizziness

Feature Vestibular (Inner Ear) Cervicogenic (Neck)
Type True rotational vertigo Vague lightheadedness
Trigger Head position changes Neck movement specifically
Nystagmus Present Absent
Key test Dix-Hallpike, head impulse Cervical torsion test
Duration Seconds to hours Minutes to hours, linked to neck position
Associated Nausea, hearing changes Neck pain, headache

Important: In Marshall syndrome, both can be present simultaneously. The cervical contribution may be more autonomic/orthostatic than true cervicogenic vertigo.

Vestibular Testing for Children

Recommended tests for age 8+:

Test What It Measures Notes for 9-Year-Old
vHIT (video head impulse) Each semicircular canal Well-tolerated, 5–10 min, no dizziness
Caloric testing Low-frequency horizontal canal More intimidating, reliable from age 6–7
cVEMP (cervical) Saccular function 10–15 min, tests otolith organs for postural stability
oVEMP (ocular) Utricular function 100% reliable by age 4
Posturography Functional balance integration Shows which sensory system child relies on most
Why VEMP Testing Matters: VEMP testing is particularly important — it tests the otolith organs that control the vestibulospinal reflex (automatic anti-gravity tone). Abnormal VEMPs could directly explain “legs buckling” episodes.
Before Cochlear Implant Discussion: cVEMP should be done before any cochlear implant discussion. CI surgery has a 34% vestibular complication rate — in a child with already-compromised vestibular function, further damage could be catastrophic for balance.

Fall Risk Management

  • 2.6-fold increased odds of falling with balance dysfunction; 12-fold when clinically symptomatic
  • Non-slip surfaces, grab bars, supervised stairs
  • Activities in darkness are particularly risky (visual compensation removed)
  • Uneven terrain is dangerous (proprioceptive compensation degraded)
  • Evening fatigue degrades all compensatory mechanisms simultaneously
  • Track falls and near-falls with the symptom tracker

Vestibular Rehabilitation (Modified)

VRT works through adaptation, habituation, and substitution. Because the COL11A1 vestibular deficit is congenital and stable, VRT should be effective.

Modifications required:

  • Gaze stabilization exercises: 3x/day, 12–20 min total, can be done as a game
  • Balance training with no extreme neck movements — spinal precautions apply
  • Aquatic balance training is ideal — vestibular challenge with zero fall risk
  • Avoid rapid head/neck movements that could worsen cervical instability
  • Video game-based exercises improve compliance in children

The vestibulosympathetic system connects the inner ear to autonomic regulation — VRT may improve autonomic symptoms too (86% improvement in one study).

What Should Be Done

  1. Baseline vestibular assessment at diagnosis (vHIT + cVEMP + oVEMP, same visit as audiogram)
  2. Repeat if hearing changes — worsening hearing may indicate worsening vestibular function
  3. Posturography if balance complaints emerge
  4. Mandatory vestibular assessment before any cochlear implant surgery
  5. Integration with spinal monitoring — new balance problems need both vestibular and cervical evaluation
  6. Modified vestibular rehabilitation respecting spinal precautions

Questions for Doctors

  1. “Can we add vestibular testing (vHIT, VEMP, posturography) to the next audiology visit?”
  2. “The Griffith 2000 study found subclinical vestibular dysfunction in Marshall patients — has this been evaluated?”
  3. “Can we do a cervical torsion test to help distinguish vestibular from cervicogenic dizziness?”
  4. “Would modified vestibular rehabilitation be appropriate given the spinal precautions?”
